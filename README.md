# CIF-Format-Stix

## INSTALLATION

To install this module type the following:
```
    perl Makefile.PL
    make
    make test
    make install
```
## DEPENDENCIES
```
    python-stix library (sudo pip-2.7 install stix)
    libxml-dev
    libxslt1-dev
    cython
```
